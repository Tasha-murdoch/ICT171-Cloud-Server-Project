For my project I have chosen to use DigitalOcean to base my cloud server on.

The steps that i took to do this include:

1. Creating a digital ocean account
<img width="346" height="340" alt="image" src="https://github.com/user-attachments/assets/e72f1113-26ac-4740-a68e-c45cb6969b9e" />

2. Create a new Droplet, this will be your cloud server. You can decide on which droplet to get depending on your system requirements.

3. Ensure that Ubuntu is picked when your are choosing your droplet as well.

4. Select the region that is most suitable to your area.

5. Add an SSH key for more security.

6. After this you have created a cloud server.

From doing this a cloud server should be created and an IP address should be assigned to the server. This is available to view under your droplets in the left side tab.

To encrypt my website I chose to use Let's Encrypt as it offers a free SSL certificate.
When installing this i used the following command

apt install certbot python3-certbot-apache -y

This installed certbot and the aoache plugin needed to get the certificate. Certbot helps to verify domain ownership and install certificates.
This is also done through apache.

The SSL certificate is then generated through Let's Encrypt

certbot --apache

After choosing my current domain name for my server the certificate then applied to my webpage.
This could be confirmed by clicking on the lock icon next to the URL of my server.

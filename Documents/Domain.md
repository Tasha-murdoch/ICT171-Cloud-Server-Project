The purpose of a domain name is so that it makes your website more easily recogniseable and memorable. INstead of needing to memorise a IP address it is much more simple to remember a named website.

The process to create a domain name for a website is as below -

1. Open Namecheap or any other relevant domain name websites
<img width="1821" height="1066" alt="Screenshot 2026-05-27 141402" src="https://github.com/user-attachments/assets/20e7a9ce-c57b-4a16-8693-7fb97c8ee727" />

2. From this point you can click on the domain option on the upper left part of the page and click "Domain Name Search". From this point type in the name of the website you want and pick a domain option that matches your budget.
<img width="1534" height="862" alt="image" src="https://github.com/user-attachments/assets/e6fcd30a-3307-4d2f-9ed2-264c991433f1" />

3. Once you have decided on a domain, add it to the cart and purches it with the subscriptions that you choose.
<img width="1676" height="930" alt="image" src="https://github.com/user-attachments/assets/2cc03393-14b5-412f-8884-17c2046d13a0" />

4. After purchasing the domain the next step is to connect it to your cloud server. To do this you go into the manage section of your domain and then click "Advanced DNS"
<img width="1793" height="950" alt="image" src="https://github.com/user-attachments/assets/938c3be1-717b-4599-b694-b7016c69c1de" />
Once you've gone into this set up your server connection buy choosing A Record and inputting your public IP address for your server as seen in the above image.

Once this is submitted it may take a while for your domain name and server to connect but once it does it will show up with the default apache similar to the image below.
<img width="1100" height="694" alt="image" src="https://github.com/user-attachments/assets/b527f241-cb93-4ed0-b6b9-ffe70b397dd8" />

Once you see the default screen and your server shows up under the domain name of your choosing, you have successfully connected your IP server address to your domain name.

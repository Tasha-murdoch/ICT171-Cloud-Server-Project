The purpose of a domain name is so that it makes your website more easily recogniseable and memorable. Instead of needing to memorise a IP address it is much more simple to remember a named website.

The process i used to create a domain name for a website is as below -

1. I chose Namecheap as my option to buy a domain name, any other relevant domain name websites do the same.
<img width="1821" height="1066" alt="Screenshot 2026-05-27 141402" src="https://github.com/user-attachments/assets/20e7a9ce-c57b-4a16-8693-7fb97c8ee727" />

2. From start webpage I clicked on the domain option on the upper left part of the page and then clicked "Domain Name Search". From this point I typed in the name of the website I wanted picked a domain option that matches my budget.
<img width="1589" height="869" alt="image" src="https://github.com/user-attachments/assets/ade8ff80-b2d2-417f-a367-5c5af4feb119" />


3. Once I decided on a domain, I added it to the cart and purchased it with the no added prescriptions as it was not necessary for my server.
<img width="1676" height="930" alt="image" src="https://github.com/user-attachments/assets/2cc03393-14b5-412f-8884-17c2046d13a0" />

4. After purchasing the domain the next step I too was to connect it to my cloud server. To do this I went into the manage section of my domain and then clicked "Advanced DNS".
<img width="1793" height="950" alt="image" src="https://github.com/user-attachments/assets/938c3be1-717b-4599-b694-b7016c69c1de" />
In this i set up my server connection buy choosing A Record and inputting my public IP address for my server as seen in the above image.

Once I submitted this it took a while for my domain name and server to connect but once it did, it showed up with the default apache similar to the image below.
<img width="1100" height="694" alt="image" src="https://github.com/user-attachments/assets/b527f241-cb93-4ed0-b6b9-ffe70b397dd8" />

Once the default screen and my server showed up under the domain name of my choosing, I successfully connected my IP server address to my domain name.

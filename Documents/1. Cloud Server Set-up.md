For my project I have chosen to use DigitalOcean to base my cloud server on.

The steps that i took to do this include:

1. Creating a digital ocean account
<img width="346" height="340" alt="image" src="https://github.com/user-attachments/assets/e72f1113-26ac-4740-a68e-c45cb6969b9e" />

2. Create a new Droplet, this will be your cloud server. You can decide on which droplet to get depending on your system requirements.
<img width="1803" height="978" alt="image" src="https://github.com/user-attachments/assets/9b3ccf0a-139a-4490-a42a-9e132c45abe8" />
<img width="1814" height="1005" alt="image" src="https://github.com/user-attachments/assets/c5e8a1da-a53b-4ac5-99f7-5f06fd9f6039" />
Ensure that Ubuntu is picked when your are choosing your droplet as well.


4. Select the region that is most suitable to your area.
<img width="858" height="299" alt="image" src="https://github.com/user-attachments/assets/447fa718-a329-4881-93f5-09d441c05d76" />


6. Add an SSH key for more security.
<img width="821" height="266" alt="image" src="https://github.com/user-attachments/assets/ce0684da-aa6b-4f36-8e7b-8fa951f885d4" />


7. After this you have created a cloud server and it should be displayed on your homepage under projects.
<img width="454" height="380" alt="image" src="https://github.com/user-attachments/assets/8a10cbac-5eaf-43b7-84dd-ecd02cb3df80" />


From doing this a cloud server should be created and an IP address should be assigned to the server. This is available to view under your droplets in the left side tab.

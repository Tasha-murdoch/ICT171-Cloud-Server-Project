Apache is an open-source web server software that I used in my assignment to implement my web server.
Apache was installed using the following command:

apt install apache2 -y


The Apache service status was verified using:

systemctl status apache2
<img width="1301" height="461" alt="image" src="https://github.com/user-attachments/assets/13240155-f774-4de9-ba40-ea61ad96a6b0" />
This confirms that the service is active and running.


The default apache webpage should be visible now when you input your server IP into the URL bar.

To make direct edits to this page the following command will access the websites files allowing you to change the current webpage.

/var/www/html
